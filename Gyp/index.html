Data Model

Structured Around Your Business

Clean entity relationships with audit trails, automated workflows and full history tracking.

👥
Customer Entity
Organization name, contact person, customer type classification, status and complete audit trail. Track relationship history over time.
📁
Project Entity
Link each project to customers. Track project names, descriptions, categories and relationships to licenses and sales.
🔑
License Entity
Type (perpetual/subscription/rental), status (active/expired/suspended), version control, start/end dates and expiration alerts.
💳
Payment Entity
Amount, currency, date paid, payment method, confirmation status and customer links for complete financial tracking.
📊
Sale Entity
Sale date, type (one-time/installment/subscription), amount, currency and customer/project links for billing management.
🎫
Service Request
Category, priority, status workflow (pending → in progress → resolved), description, attachments and complete history.
Development Roadmap

From MVP to SaaS Platform

Structured 6-phase implementation plan spanning 12–15 weeks to build a mature, scalable product.

Phase 1
Core Infrastructure (Weeks 1–3)
ABP solution setup, Next.js configuration, OpenIddict authentication, Docker deployment, database setup. Deliverable: Login/logout working, authenticated screens accessible.
Phase 2
Customer & Project Mgmt (Weeks 4–6)
Complete CRUD operations, list views, detail pages, filtering and pagination. Deliverable: Full customer and project management with basic UI.
Phase 3
License Tracking (Weeks 7–8)
License lifecycle, status management, automated expiration tracking with email alerts. Deliverable: Full license management with notifications.
Phase 4
Payment & Debt (Weeks 9–11)
Payment plans, recording, debt calculation, overdue tracking. Dashboard with receivables, collected amounts and risk indicators.
Phase 5
Service Requests (Weeks 12–13)
Request creation, category/priority management, status workflow, attachments. Full tracking system with workflow automation.
Phase 6
Polish & SaaS (Weeks 14–15)
Multi-tenant preparation, advanced search, CSV export, performance optimization. Deliverable: Production-ready platform ready for customer rollout.
