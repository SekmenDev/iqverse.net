## About

Favicon & Meta Generator is a modern, browser-based tool for creating complete favicon sets and HTML meta tags. Every generation is:

- **100% Client-Side** – Your images never leave your browser
- **Open Source** – MIT licensed, source code on GitHub
- **Privacy-First** – No tracking, no telemetry, no accounts needed
- **Developer-Friendly** – Built by developers, for developers

---

## ✨ Features

- Generate favicons from PNG, SVG or JPG images
- Automatic resizing to standard sizes (16x16, 32x32, 180x180, 192x192, 512x512)
- Complete HTML `<head>` code with meta tags
- Open Graph and Twitter Card support
- Theme color and language settings
- Export as data URIs or file paths
- Real-time preview of generated icons
