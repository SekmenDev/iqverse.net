Link Radar

Scan any webpage and detect broken links in seconds. Paste a URL or raw HTML and Link Radar will crawl all anchors and report 404s, timeouts and redirect chains. So you can fix them before users hit them.

  • Bulk link scanning
  • HTTP status detection
  • Redirect chain analysis
  • Exportable reports

Favicon Generator

Upload any image or design a favicon from scratch. Automatically exports all sizes needed for modern browsers: 16×16, 32×32, 180×180 Apple touch and the full manifest.json. No Photoshop required.

  • All favicon sizes in one click
  • SVG favicon support
  • Web manifest generation
  • Browser preview
