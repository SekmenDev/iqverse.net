## About

IQVerse is a collection of modern, browser-based developer tools. Every tool is:

- **100% Client-Side** – Your data never leaves your browser
- **Open Source** – MIT licensed, source code on GitHub
- **Privacy-First** – No tracking, no telemetry, no accounts needed
- **Developer-Friendly** – Built by developers, for developers

---

## 🚀 Getting Started

All tools run entirely in your browser. No installation required.

1. **Visit a tool** – Click any link above to launch
2. **Use it immediately** – No login, no setup
3. **Your data stays private** – Everything processed locally

## 🤝 Contributing

All tools are open source and MIT licensed. We welcome:

- 🐛 Bug reports and fixes
- ✨ Feature requests and implementations
- 💡 New tool ideas
- 📚 Documentation improvements

### Steps to Contribute

1. Fork the repository of the tool you want to improve
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to your branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request
