# IQVerse Tools

> **Free. Open. Powerful.**  
> Tools built by developers, for developers. No logins. No paywalls. No telemetry.

![License](https://img.shields.io/badge/license-MIT-green) ![Open Source](https://img.shields.io/badge/open-source-blue) ![Browser](https://img.shields.io/badge/browser-based-9cf)

## About

IQVerse is a collection of modern, browser-based developer tools. Every tool is:

- **100% Client-Side** – Your data never leaves your browser
- **Open Source** – MIT licensed, source code on GitHub
- **Privacy-First** – No tracking, no telemetry, no accounts needed
- **Developer-Friendly** – Built by developers, for developers

---

## 🛠️ Available Tools

### [QR Forge](https://qrforge.iqverse.net/) ⚡ Live

Generate QR codes instantly in your browser. Fast, powerful and zero data collection.

**Features:**

- Generate QR codes from URLs, text, Wi-Fi credentials and vCards
- 100% client-side generation
- Export as SVG or PNG at any resolution
- Custom colors and logo support

📍 [Launch Tool](https://qrforge.iqverse.net/) · [Source Code](https://github.com/SekmenDev/qr-forge)

---

### [Link Radar](https://linkradar.iqverse.net/) ⚡ Live

Scan any webpage and detect broken links in seconds. Crawl all anchors and identify 404s before your users do.

**Features:**

- Bulk link scanning from URLs or raw HTML
- HTTP status detection & redirect chain analysis
- Exportable reports
- Real-time feedback

📍 [Launch Tool](https://linkradar.iqverse.net/) · [Source Code](https://github.com/SekmenDev/link-radar)

---

### [School Management](https://goo.iqverse.net/) 🎓 Live

A full-featured school management platform trusted by educational institutions.

**Features:**

- Student & staff records management
- Real-time attendance tracking
- Grade management & digital report cards
- Integrated parent portal
- Privacy-first design

📍 [Visit App](https://goo.iqverse.net/) · _Proprietary SaaS_
