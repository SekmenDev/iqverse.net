01 Open Source Live

QR Forge

Generate QR codes instantly in your browser without any data collection. Supports URLs, plain text, Wi-Fi credentials, vCards and more. Export as SVG or PNG at any resolution.

  • 100% client-side generation
  • SVG & PNG export
  • Custom colors & logos
  • Multiple data types

02 Open Source Live

Link Radar

Scan any webpage and detect broken links in seconds. Paste a URL or raw HTML and Link Radar will crawl all anchors and report 404s, timeouts and redirect chains. So you can fix them before users hit them.

  • Bulk link scanning
  • HTTP status detection
  • Redirect chain analysis
  • Exportable reports
